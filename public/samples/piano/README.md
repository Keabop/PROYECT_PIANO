# Muestras de piano

Muestras del **Salamander Grand Piano** (Yamaha C5) grabado por **Alexander Holm**,
licencia [CC BY 3.0](https://creativecommons.org/licenses/by/3.0/).
Fuente: https://archive.org/details/SalamanderGrandPianoV3 (vía tonejs.github.io/audio/salamander).

Se incluye un subconjunto (una muestra cada tercera menor, C2–C7); el resto de notas
se interpola por pitch-shifting con Tone.Sampler.
